# LumiControl

无线遥控器 — 把手机变成投影仪 / 智能电视 / 安卓设备的触控板和遥控器。

## 适用场景

| 场景 | 说明 |
|------|------|
| 投影仪无遥控器 | 手机扫码即可控制，触控板滑动移动光标，点击确认 |
| 安卓智能电视 | 同一 WiFi 下手机当遥控器，支持方向键、主页、返回、菜单 |
| 安卓手机 / 平板被控 | 在一台安卓设备上安装 LumiControl，另一台手机通过网页控制它 |
| 安卓盒子 / 游戏主机 | 只要运行 Android 系统，均可通过浏览器遥控 |
| 会议演示 | 手机当无线鼠标 + 键盘，远程操控投影内容 |

## 功能

- **触控板** — 滑动移动光标，轻触点击，支持手势操作
- **摇杆** — 四方向滚轮，精准翻页
- **遥控器** — 主页 / 返回 / 菜单 / 音量 / 设置 / 复位
- **文字输入** — 手机端输入文字，直接发送到被控设备
- **光标自定义** — 圆形 / 圆点 / 箭头 / 十字 / 自定义 PNG
- **APK 安装** — 通过网页上传 APK 到被控设备安装
- **自动发现** — 同一局域网内自动发现设备，扫码连接

## 使用方式

### 被控端（投影仪 / 电视 / 安卓手机）

1. 安装 LumiControl APK
2. 打开应用，授予无障碍权限和悬浮窗权限
3. 屏幕显示设备 IP 地址和二维码
4. 保持应用在后台运行

### 控制端（任意手机浏览器）

1. 确保两台设备在同一 WiFi 下
2. 打开被控端显示的网址（如 `http://192.168.1.100:4560`）
3. 或扫描被控端屏幕上的二维码
4. 开始操控

### 控制安卓手机

LumiControl 可以安装在任何安卓设备上，因此你可以：

1. 在要被控制的安卓手机上安装 LumiControl
2. 用另一台手机的浏览器打开该手机的 IP 地址
3. 实现远程控制安卓手机

## 技术栈

- Android Kotlin + NanoHTTPD
- WebSocket 实时通信
- 无障碍服务（AccessibilityService）实现光标控制和按键注入
- 纯前端 HTML/CSS/JS，无需安装控制端 App
